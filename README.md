# comoresopendata-data

Data processing pipeline for [comoresopendata.org](https://comoresopendata.org).

This repository contains scripts to extract, clean and export datasets from international sources (IATI, World Bank, etc.) for publication on the Comoros open data platform.

## Structure

```
sources/
  iati/         ← IATI international aid data (d-portal API)
  worldbank/    ← World Bank indicators (coming soon)
outputs/        ← Final Excel/CSV files ready to upload (gitignored, regenerate locally)
```

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run IATI extraction

```bash
python3 sources/iati/iati_data_extraction.py
```

Output: `outputs/comores_aide_internationale.xlsx`

## Data sources

- **IATI:** https://d-portal.org/q.json — all international aid activities for Comoros (country code: KM)
- **World Bank:** coming soon

## Notes

- `outputs/` is gitignored — files are large and can be regenerated by running the scripts.
- Multinational activities (where KM appears alongside other recipient countries) are included in the extraction but flagged with `activite_multinationale = Oui`.
- d-portal's country page (https://d-portal.iatistandard.org/ctrack.html?country=KM) shows only the 1,016 non-multinational activities. Our dataset includes 1,273 activities total.
